# Credit Card Fraud Detection

Minimal transaction risk analysis app built with Streamlit and SHAP explanations.

## Run

Install dependencies, then start the app:

```bash
streamlit run app.py
```

## Required Local Files

Place these before running:

- `data/raw/fraudTrain.csv`
- `data/raw/fraudTest.csv`
- `data/processed/data.pkl`
- `data/processed/scaler.pkl`
- `models/random_forest.pkl`
- `models/stacking_model.pkl`

These files are not stored in the GitHub repo.
